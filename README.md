## ğŸ¤– Machine Learning Model

### **Algorithm: KMeans Clustering**

#### **Why KMeans?**
- Fast and efficient for large datasets
- Interpretable results for business stakeholders
- Well-suited for numerical customer data
- Scalable to production environments

### **Feature Engineering**

We create **16+ engineered features** from raw data:

#### **1. Product Spending Features**
```python
Total_Product_Spending = Wines + Fruits + Meat + Fish + Sweets + Gold
Premium_Product_Ratio = (Wines + Meat) / Total_Product_Spending
Budget_Product_Ratio = (Fruits + Sweets) / Total_Product_Spending
```

#### **2. Purchase Channel Features**
```python
Total_Purchases = Web + Catalog + Store
Online_Ratio = Web / Total_Purchases
Store_Ratio = Store / Total_Purchases
```

#### **3. Customer Behavior Features**
```python
Purchase_Frequency = Total_Purchases / Days_as_Customer
Avg_Purchase_Value = Total_Spending / Total_Purchases
Days_Per_Purchase = Days_as_Customer / Total_Purchases
```

#### **4. Engagement Features**
```python
Promo_Acceptance_Rate = Total_Promo / Total_Purchases
Discount_Ratio = Discount_Purchases / Total_Purchases
Web_Engagement = NumWebVisitsMonth / 30
```

#### **5. RFM-like Features**
```python
Recency_Score = 100 - Recency
Monetary_Score = Total_Spending
Frequency_Score = Total_Purchases
```

#### **6. Customer Value**
```python
Customer_Lifetime_Value = Total_Spending * (Days_as_Customer / 365)
Income_to_Spending_Ratio = Total_Spending / Income
```

### **Model Training Pipeline**

```python
1. Generate/Load Data
   â†“

   â†“

   â†“
```

### **Model Evaluation**

We use **3 comprehensive metrics**:

1. **Silhouette Score** (0.4-0.6)
   - Measures cluster cohesion
   - Range: -1 to 1 (higher is better)
   - Our model: ~0.45

2. **Davies-Bouldin Index** (0.8-1.2)
   - Measures cluster separation
   - Lower is better
   - Our model: ~0.9

3. **Calinski-Harabasz Score** (1000-2000)
   - Variance ratio criterion
   - Higher is better
   - Our model: ~1500

### **Customer Segments**

#### **Cluster 0: Budget-Conscious Shoppers**
- High discount usage
- Lower spending
- Price-sensitive
- **Strategy**: Discount campaigns, loyalty programs

#### **Cluster 1: Premium Customers**
- High income & spending
- Premium products
- Brand loyal
